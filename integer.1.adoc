= integer(1)

== Name

integer - check if all given arguments are non-negative integers

== Synopsis

*integer* _STRING_...

== Exit status

*0*::
    All strings given were non-negative integers.

*1*::
    At least one string was not a non-negative integer.

== Notes

For the purposes of scripting, 0 (zero) is considered a non-negative integer.
